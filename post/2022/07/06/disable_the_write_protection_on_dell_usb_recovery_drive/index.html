如何格式化一个写保护的戴尔 (DELL) 恢复 U 盘

2022-07-06
其他
约 926 字 预计阅读 2 分钟

文章目录

大学时买的是戴尔 (DELL) 笔记本，后来申请了个恢复 U 盘（见下图），里边是 Windows 8.1 系统镜像。去年翻出来想格式化，却因为写保护而作罢，最近想起来再折腾一下就找到方法了😀。

方法在 2016 年的一个技术文章中（互联网考古）：http://www.techunboxed.com/2016/09/how-to-disable-write-protection-on-dell.html ，用到的工具在原文的 Mega 网盘中，而我在 Github 找到了 ( https://github.com/amgxv/DellMentorMediaUSBFormat )。

下载工具后，插入 U 盘（记下接口 USB 是 2.0 还是 3.0 ，后面要用到），打开名为 “GetInfo v3.10.7.6” 的程序，输入盘符，点击 “read” 获取 U 盘信息（我的这个戴尔恢复 U 盘已经格式化成功，只作演示）：

再打开名为 “MPALL_F1_7F00_DL07_v503_0A” 的程序，点击 “Update” 按钮，界面上会出现插入的 U 盘设备：

确定 U 盘已连接后，点击 “Setting” 按钮：

保持默认的 “Basic Setting”、“New Setting” 就行，接着点击 “OK”。

进入如下窗口：

在此窗口中，点击 “Controller” 下拉框，选择 “PS2251-07”，在 “Host Port” 下拉框中选择之前插入 U 盘时的接口类型（2.0 或 3.0）；勾选 “Do ISP” 和 “Erase code block”，接着点击 “Select”，在 “MPALL_F1_7F00_DL07_v503_0A” 文件夹中选择名为 “BN07V502TAW.BIN” 的烧录文件（burner file），再点击下一行的 “Select”，选择名为 “FW07V50253TW.BIN” 的固件文件（firmware file）。

下一步，在 “Customise Setting” 中，去掉 “Auto Detect” 的勾选框，分别输入前面获取的 VID、PID。设置完成后，应如下图所示：

接着点击 “Save” 按钮保存设置或者 “Save As” 按钮将以上设置保存到另一个配置文件以便下次直接加载配置文件使用；保存完设置关闭窗口后，回到主程序窗口，此时若 U 盘 那是灰色或者未显示出来，再点击一次 “Update” 就刷新出来了。

一切准备就绪，点击 “Start” 按钮，大概几分钟后，U 盘显示为绿色时就说明格式化成功，原来的写保护已被去除：
