收集整理关于互联网中个人隐私保护的相关资源，主要以开源为主：

资源

Privacy Guides

官网：https://www.privacyguides.org/ , Github

网站收集注重安全、隐私的资源，包括操作系统、各种应用软件。

注：此项目由 PrivacyTool（即下文所指）迁移而来；Github上的原项目 (PrivacyTool.io) 由于成员内讧已变为归档状态，其他项目成员转而维护新项目(Privacy Guides)

Privacy Tools

官网：https://www.privacytools.io/

网站收集注重安全、隐私的资源，包括操作系统、各种应用软件。

Privacy Tools List

官网：https://privacytoolslist.com/

网站收集注重安全、隐私的资源，包括操作系统、各种应用软件。

Awesome Privacy

Github：https://github.com/pluja/awesome-privacy

收集注重隐私保护的相关服务、应用软件。

Degoogle

Github：https://github.com/tycrek/degoogle

收集可替代谷歌全家桶的应用软件。

Digital-Privacy

Github：https://github.com/ffffffff0x/Digital-Privacy

关于隐私收集、保护、清理的方案。

工具

WindowsSpyBlocker

设置防火墙策略，屏蔽相关 IP 连接

官网：https://crazymax.dev/WindowsSpyBlocker/ , Github

Privatezilla

设置策略组等屏蔽 Windows 数据收集

官网：https://www.builtbybel.com/ms-apps/privatezilla , Github

其他

诸如 WPD,W10Privacy 等免费但未开源的工具也可，看个人选择。

WindowsSpyBlocker、Privatezilla 对我来说已足够了。
