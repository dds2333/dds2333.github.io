WEB 浏览器的选择

2021-11-21
其他
约 654 字 预计阅读 2 分钟

文章目录

当微软推出基于 “Chromium” 的 WEB 浏览器 - “Microsoft Edge” 时，估计很多人跟我一样直呼真香，而且使用体验完全不输 “404 公司”（Google）。

然而当积累足够多的用户时，微软的做法却向 “404 公司” 趋近：Edge 新标签页出现关不掉的广告（当时据称为测试，现已去掉，将来会不会再加入未知）、强制同步用户收藏夹以及 WIN 11上更换默认浏览器的繁琐操作。如果使用必应搜索并登录微软账户，则在微软账户管理的“隐私”中会看到搜索记录。

因此我找了一些可替代 Edge 的浏览器：

  • Chrome （讲个笑话，Google 不作恶，直接不考虑）
  • Firefox （勉强可以，开源，但有烦人的更新提醒）
  • Brave （不做评价，官网需富强）
  • Waterfox （开源，64 位的 Firefox 分支，去掉了 Firefox 的一些追踪器但也有烦人更新提醒）
  • Iridium Browser （开源，基于 Chromium ，去掉了 Google 的相关追踪器）
  • ungoogled-chromium （开源，去掉了 Google 的相关追踪器，最近更新了 v96 的 Windows 版本 ）
  • LibreWolf（开源，Firefox 分支，无烦人的更新提醒，增强的隐私设置）
  • 其他不一一列举

国内比如 360 浏览器之类的，大多一堆辣鸡广告，不考虑。

比较下来，我选择了 ungoogled-chromium / Iridium Browser、LibreWolf，LibreWolf 作为主力，不过由于其隐私设置严格，有些网站可能会出现无法加载图片等问题，因此再辅以 ungoogled-chromium / Iridium Browser，而且这三个浏览器更新还比较及时。

这三个浏览器其他都还好，就是同步收藏夹比较麻烦，这也是预料之中吧，辩证唯物主义😎。

最后给浏览器装上 uBlock Origin、ClearURLs 等扩展，美滋滋😁。
